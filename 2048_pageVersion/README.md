# 网页版的2048

这个网页版的2048游戏是来自[source](http://bbs.csdn.net/topics/390763959);


## 修改说明

第一：添加了游戏结束判断逻辑（源代码中存在比较大的bug，他的逻辑只是判断当前方向上是否存在合并）
```
   主要思想是：
              1、 如果空块数量为0个，跳转步骤2；否则，游戏继续；
			  2、 每个方向上都进行判断是否有合并的块，因上下和左右方向类似，所以只需要判断两个方向；
			      这里假设为左上，并把每个方向合并的可能值加入临时数组（总共八个）；
			  3、 如果数组的值全部为true则说明Game Over，否则游戏继续；
			  
```
